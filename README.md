# The Adult Swim Singles Program

Adult Swim curated and released free music for over a decade though *The Adult Swim Singles Program* and Williams Street Records. Each year of free music was released through its own web page with colorful psychedelic animations. This page indexes every song released as an "Adult Swim Single" since 2010.
